        <span>Executive Summary</span>
        <span class="accordion-icon">+</span>
      </button>
      <div class="accordion-content">
        <p><strong>Atomic Habits</strong> is not really a book about habits. It is a book about how change actually happens in real life. James Clear argues that most people fail not because they lack ambition, discipline, or intelligence, but because they rely too heavily on goals and too little on systems.</p>
        <p>His core message is deceptively simple: if you improve your routines in tiny, repeatable ways, the results eventually compound into dramatic transformation.</p>
        <p>The book’s power lies in how practical it is. Rather than telling people to “want success more,” it shows how to design behavior so that success becomes more likely by default.</p>
        <p>Its most important contribution is the shift from outcome-based thinking to identity-based behavior. Instead of asking, “What do I want to achieve?”, Clear pushes readers to ask, “Who do I want to become?”</p>
        <p>For leaders and operators, the book is especially useful because it scales beyond personal life. Small repeated behaviors—meeting hygiene, communication rituals, review cadence, feedback loops, documentation discipline—compound into either operational excellence or chronic dysfunction.</p>
      </div>
    </div>

    <div class="accordion-subsection">
      <button class="accordion-toggle level-2" type="button" aria-expanded="false">
        <span>Background of the Book</span>
        <span class="accordion-icon">+</span>
      </button>
      <div class="accordion-content">
        <p>James Clear wrote <strong>Atomic Habits</strong> after years of writing about decision-making, behavior change, and continuous improvement on his website and newsletter.</p>
        <p>The book emerged from a practical question: Why do smart, motivated people still fail to change consistently?</p>
        <p>His answer is that behavior is usually a design problem, not a character flaw.</p>

      <div class="accordion-content">
        <p>For leaders, the biggest lesson is this: <strong>culture is just collective habit.</strong></p>
        <p>Organizations rarely fail because of one dramatic strategic mistake. They fail because small repeated behaviors create drag: unclear ownership, sloppy follow-up, meeting overload, weak review loops, and tolerance for mediocrity.</p>
        <ul>
          <li>Make good behaviors obvious → dashboards, rituals, visible expectations</li>
          <li>Make them attractive → recognition, social proof, incentives</li>
          <li>Make them easy → remove bureaucracy and ambiguity</li>
          <li>Make them satisfying → close loops, celebrate wins, show progress</li>

document.addEventListener("DOMContentLoaded", function () {

  function setHeightAutoThenScroll(content) {
    if (!content) return;

    // Reset first (important)
    content.style.maxHeight = "none";

    // Force reflow
    content.offsetHeight;

    // Then set correct height
    content.style.maxHeight = content.scrollHeight + "px";
  }

  function updateAllParents(element) {
    let current = element.parentElement;

    while (current) {
      if (
        current.classList &&
        (current.classList.contains("accordion-section") ||
         current.classList.contains("accordion-subsection"))
      ) {
        const parentContent = current.querySelector(":scope > .accordion-content");

        if (parentContent && current.classList.contains("open")) {
          setHeightAutoThenScroll(parentContent);
        }
      }
      current = current.parentElement;
    }
  }

  const accordionToggles = document.querySelectorAll(".accordion-toggle");

  accordionToggles.forEach(toggle => {
    toggle.addEventListener("click", function () {
      const parent = this.closest(".accordion-section, .accordion-subsection");
      const isOpen = parent.classList.contains("open");
      const content = this.nextElementSibling;

      parent.classList.toggle("open");
      this.setAttribute("aria-expanded", String(!isOpen));

      if (content) {
        if (!isOpen) {
          // Step 1: expand child FIRST
          setHeightAutoThenScroll(content);

          // Step 2: wait for DOM paint, then update parents
          requestAnimationFrame(() => {
            updateAllParents(parent);
          });

        } else {
          content.style.maxHeight = null;

          requestAnimationFrame(() => {
            updateAllParents(parent);
          });
        }
      }
    });
  });

  // Initial open sections
  document.querySelectorAll(
    ".accordion-section.open > .accordion-content, .accordion-subsection.open > .accordion-content"
  ).forEach(content => {
    setHeightAutoThenScroll(content);
  });

  // Resize fix
  window.addEventListener("resize", () => {
    document.querySelectorAll(
      ".accordion-section.open > .accordion-content, .accordion-subsection.open > .accordion-content"
    ).forEach(content => {
      setHeightAutoThenScroll(content);
    });
  });

});
